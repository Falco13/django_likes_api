# django_likes_api
# Django application with Posts and Like's for it

- A simple django app with the ability to Like all Posts.


__API end-points:__
- api/posts/
- api/posts/id
- api/posts/id/like
- api-auth/


__Used tools:__    
:heavy_check_mark: Python     
:heavy_check_mark: Django REST Framework      
:heavy_check_mark: SQLite database
